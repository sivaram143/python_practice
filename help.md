### Help

```
$ python3  # To open python interpreter

>>> help() # Move to help prompt
help> modules # list the modules
help> symbols # list the symbols
help> keywords # list the keywords
help> topics # list the topic
help> quit   # to close the help prompt
>>> dir() # show the names in the module namespace
>>> dir(<module/object>) # show the properties/methods in the module/objects

Note: Type name of module/topic/symbol/keywords to get more help of particular.
```
